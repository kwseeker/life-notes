## 基金基础导读

+ 基金就是为某种目的而设立的一笔资金。比如我们知道的公积金、社保基金、风险投资基金都属于广义的基金范畴。狭义上的基金，一般就是指公募证券投资基金，简称公募基金。

+ 收集散户投资者资金交由专业投资人士去投资证券市场赚取收益。
+ 公募基金，也有多种风险等级，购买门槛低。
+ 目前我国的基金总数超过4800支，基金公司超过100家（2017年的数据）。

+ 基金命名的规则一般是：基金公司+投资方向+基金类别+收费类型。

  > 基金类别：债券型基金？混合型基金？大盘股？小盘股？
  >
  > 投资方向：沪深300指数？中小板？创业板？
  >
  > 收费类型：前端收费模式？
  >
  > 重仓股票？

  基金按资产类别分类：

  + **货币型基金**（风险很低、收益也低，但是高于活期存款）

    专门投资风险小的货币市场，比如我们熟知的国债、央行票据、商业票据、银行定期存款、同业存款等。

    常用于存放应急备用金。

    相当于拿着定期存款的收益，享受活期存款的灵活性。

    细分类型：

    + **A类**

    + **B类**（起始投资金额500万元或1000万元）

      但某些公司将B类基金购买门槛降到了100元起购。

    优越衡量标准：

    + **万份收益率**

      体现的是基金每天的收益变化，遇到月底、年底市场缺钱的时候，万份收益率会波动很大。

    + **七天年化收益率**

      七天年化收益率把短期的波动因素考虑进去，平均了过去7天的收益变化。

    特点：

    + 资金流动性强

    + 投资成本低

      一般都免收手续费，认购费、申购费、赎回费都为0。

    + 方便转换

      股市好的时候可以转成股票基金，债市好的时候可以转成债券型基金，当股市、债市都没有很好机会的时候，货币市场基金则是良好的资金蓄水池。

    + 与市场资金面成反比

      市场缺钱，收益率会变高，市场资金宽松，收益率降低。

      一般在月末、季度末和年底的时候，市场资金面相对紧张，货币型基金收益率会显著提升。

      市场上资金面的松紧程度，大家可以在上海银行间同业拆放利率网站查看每天的Shibor走势图。

    > 货币型基金的资产净值是固定不变的？
    >
    > 一般都不会保证本金的安全，但事实上货币型基金的性质决定了它极少会发生本金的亏损?
    >
    > 货币型基金具有高安全性、高流动性的特点?

  + **债券型基金**（风险增加、收益增加）

    按发行方分：

    + **国债**
    + **地方债**
    + **金融债**
    + **企业债**

    > 国家为何要发行国债？
    >
    > 一、募集资金进行基础设施建设等，二、调节市场通行货币量，防止通货膨胀、调节货币利率。

    按资金投资方向配比分：

    + **纯债基金**（100%投资债券）

    + **普通债券型基金**（至少80%投资债券）

      + **一级债券型基金**

        在纯债基金的基础上，可以适当参与新股申购。

      + **二级债券型基金**

        最多可以拿出20%的资产参与投资股票。

      + **可转债基金**

        主要投资可转债。这类债券型基金的特点就是债券在特定的时候可以转化为股票，具有股票和债券双重性质。

    收益可能越来越高，伴随的风险也越来越大。

    在过去近十年（2017年数据），债券型基金的年化收益率是8.1%。

    债券型基金也会有剧烈的波动，这在很大程度上是由债券型基金购买股票等涨跌明显的资产导致的。

    债券基金入手时机：

    + 与货币政策反向债券的价格总体上是和利率成反比的
    + 与股市反向股票市场和债券市场长期以来呈现出此消彼长的关系

  + **股票型基金**（风险较高、收益较高）

    股票型基金中的股票不能低于80%。股票型基金是各基金类别中风险系数比较高的一种，同时收益也普遍较高。

    还可以按照投资的资产方向、股票种类、投资目的进行细分。

  + **混合型基金**（比较灵活）

    混合型基金的股票仓位可能只有60%，债券仓位只有20%，还有20%是货币型基金。可以同时投资股票、债券和货币，这三者在混合型基金中的比例没有严格的限制。

    + **偏股混合型基金**
    + **偏债混合型基金**
    + **股债平衡型基金**

    在市场走势不好时，基金经理可以把股票仓位降下来，从而避免损失。但是，从另外一个角度讲，混合型基金的股票仓位也可能比股票型基金的股票仓位更高，下跌起来也是一点都不含糊。

  基金按投资理念分类：

  + **被动型基金**（更通用的称呼：**指数型基金**）

    以某种特定指数为标，如上证综指、深证指数。基金名字中常带着某指数的名字，如“景顺长城*沪深300*增强”。

    被动型基金经理不用像主动型基金经理那样苦思冥想买什么股票，只需要按照构成指数的成分股配置就好了。

    长期来看，指数型基金在相关费用上的节省额也是非常可观的。

    买入指数型基金，其实是在认可市场整体长期趋势是继续向上增长的，或者说是对国家未来经济前景充满信心。

    巴菲特比较推崇购买指数型基金。

    指数基金的选择

    + **跟踪误差小**

      某支基金和指数整体趋势的拟合程度。

    + **费用更低**

    + **基金成立时间和规模**

      一定要看看基金在熊市低谷的时候是否也能稳定发挥。

      基金规模不能太小，太小的基金规模扛不住大规模赎回，尤其是在股市处于低点的时候，用户往往要求大规模赎回，小规模的基金显然招架不住。

      规模偏小的另一个尴尬点是可能会被清盘，虽然清盘不会导致亏损，但是前面辛苦积累的投资份额就归零了。

      建议大家尽量不要考虑1亿元以下规模的基金。

    指数基金分类（申购、赎回、服务费等差别）

    + **A类**

      适合长期持有，尤其是持有超过2年以上的，尤其适合那些喜欢定投指数型基金的投资者。

    + **C类**

      如果你是喜欢快进快出的人，没有申购费并且赎回费要求比较低的C类基金更适合你。

    指数型基金投资的3个关键问题

    

    > **指数？**
    >
    > 股市走向的一种趋势。
    >
    > 大家把**上证综合指数**作为股市大盘涨跌的一个重要指标。
    >
    > 上证综指的缺陷：
    >
    > 1）依据公司股票的发行量大小来设计权重，并不一定能代表全行业的波动状态。如果有一家占比较大的公司有很大的波动，可能会影响整个行业的指数导致失真。
    >
    > 2）上证综指反映的是股票交易所中所有股票的全貌，优质股票也好，垃圾股票也好，统统都算在里面。
    >
    > **沪深300指数**（大盘指数）
    >
    > 同时把上交所和深交所的股票波动情况客观地反映出来的指数。
    >
    > 选取了300支覆盖了上交所和深交所6成左右市值的股票，它能够比上证综指更全面、更真实地反映出A股市场的整体走势。可以说，沪深300指数反映了市场上大盘股的整体趋势。
    >
    > 但是注意沪深300指数反映的是整个市场中大市值公司的指数。
    >
    > **中证500指数**（中小盘指数）
    >
    > 衡量中小市值企业的指数，剔除掉了沪深300指数的所有成分股，保证这个指数和沪深300指数没有关系。之后，再把沪深300指数成分股之外的市值排名前300名的股票也剔除掉，基本保证所有大中盘股都被剔除了。最后，把剩余的股票按照市值由高到低排列，选取排名在前500名的股票作为中证500指数的成分股。
    >
    > **中小板指数**
    >
    > 包含了在深交所中小企业板上市的100支股票。
    >
    > 选取样本股票时先计算入围个股平均流通市值占市场比重和平均成交金额占市场比重，再将上述指标按2∶1的权重加权平均，计算结果从高到低排序后选取100支样本股票组成中小板指数。
    >
    > 怎么个入围规则？
    >
    > **创业板指数**
    >
    > 包含在深交所创业板上市的100支股票。这些在创业板上市的公司都是规模较大、流动性较好的股票。具体编制方法和中小板指数比较类似。
    >
    > **总结**
    >
    > 上证综指和沪深300指数主要包含蓝筹股，所以市盈率是最低的，深证成指和中证500指数的市盈率要高一点，中小板指数的更高，创业板指数最高。

  + **主动型基金**

    主动型基金需要由基金经理主动来选股、选债，试图通过人工挑选，获取比市场水平更高的收益。

    基金公司人力投入大相对申购费用高一些。

  基金按照资产地域分类：

  + **投资国内市场的基金**

  + **投资海外市场的基金**（主要是QDII基金）

    国内的基金公司可以募集投资者手中的人民币去购买国外的资产，比如股票、债券等。也就是说，我们可以利用QDII基金来配置海外的资产。

+ 基金收益来源

  资本利得（股票买卖差价）、股息红利（公司净盈利分红）、利息收入（债券利息）。

+ 基金投资误区

  + 历史业绩好就一直好吗

  + 基金越便宜越好吗

  + 明星基金就是最好的吗（明星基金的光芒都来自过去的成绩，热捧之后容易下滑）

    与其关注明星基金，不如关注基金的可持续性和稳定性。

  + 基金名字就是投资的主要方向吗

    不是所有基金都是“名副其实”的。

  + 分红的基金才是好基金吗

    分红本质上是提前减少仓位投资变现。对于持续上涨的基金分红不是好的选择。

  + 新基金一定比老基金好吗

    新上市的股票价格很有可能远远低于其价值，但是新基金投资的股票大部分是经过基金公司反复研究、不断挑选的股票，可能会有新股票在其中，但是总的来看，不会大量存在这样的情况。

    如果确实喜欢新的基金，那么次新基金可以成为投资者的一个选择。

  + 规模大的基金都是好基金吗

    基金规模大，说明选择这支基金的投资者或者机构比较多。但是，大部分的基金业绩表现会出现“规模收益递减”的特性。

  + 可以相信专家的判断吗

    市场是不可预测的。

    市场的魅力就在于它不断地考验大家，用涨跌来迷惑大家。在上涨的时候让大家信心爆棚，产生贪婪心理；在下跌的时候，让大家寝食难安，阵阵恐惧。

    夏普比率：夏普比率就是能够综合衡量收益和风险的系数，它是用基金收益减去无风险投资收益，再除以基金的标准差得出的。夏普比率高，则说明在收益相同的情况下，基金面临的风险较低；或者说在风险一定的情况下，收益较高。因此在收益或者风险不变的情况下，夏普比率越大越好。

+ 基金购买渠道

  + 银行代销网点

    基金有限、购买费用高、容易被销售误导

  + 基金公司网站

  + 第三方理财机构

    基金种类最全、手续费更低、体验更好、态度中立

  > 指数系列的基金，可以直接去广发、天弘基金官网购买；
  >
  > 量化或者增强型基金，可以直接去华泰柏瑞、景顺长城基金官网购买。

  >
  > 同时，基金公司经常会在自己的官网上推出力度很大的折扣，在费率上有一定优势。
